Patch Notes — June 29, 2026

Closed Beta final patch. Source: eqlwiki.com/Patch_notes and official Discord #natch-potes channel.

📅 Beta Timeline

Closed Beta (April 24 – June 29) concluded with this patch. Pre-Order Open Beta begins July 1 at Noon PT / 3pm ET. Characters from closed beta do not carry over to pre-order beta.

Spell Upgrade System New

Players can now upgrade spells using motes, similar to upgrading items. Right-click-hold on a spell to inspect, then add motes.

  • Upgraded spells currently gain limited benefits — more effects coming in the first week of Pre-Order Beta
  • Dev note: "Hitpoints, Damage, etc. were not completed for this patch cycle"
  • Spell upgrades do NOT reflect on item procs that are upgraded spells — those default to base

New Stance/Invocation Window

  • Dedicated window showing all learned stances and invocations
  • Hide any stance/invocation by right-clicking the button → "Hide"
  • Unhide via right-click → "Hidden" submenu
  • Customizable button colors: Stance Highlight, Stance Lasso, Invocation Highlight, Invocation Lasso
  • Every stance and invocation is now individually keybindable
  • Stance bar toggle is also bindable

Class Balance Changes

Dave's Combo (Bard/Paladin/Enchanter)

ClassChange
EnchanterEnchant Clay spell reduced to level 7 (was higher). Enchant Steel reduced to level 21 (was higher). Makes early enchanting much more accessible.
Pure MeleesCan now toggle off Unyielding Invocation. Paladin is hybrid, not pure melee — no change for Dave.
Pure CastersCan now toggle off Channeler's Stance. Enchanter is a pure caster — relevant if you run an Enchanter-primary loadout.

Christy's Combo (Cleric/Necromancer/Wizard)

ClassChange
ClericPromise Renewal reworked: 1 second cast time (was instant?), increased mana cost, heals for 5000. This is the signature Cleric heal — massive buff to burst healing but costs more mana and has a cast bar now.
ClericDuration Heal spells (Celestial Remedy, Celestial Health, Celestial Healing) — mana cost reduced.
DruidNew HoT line added: Budding Heal (L4) → Efflorescing Heal (L48). Not Christy's class, but relevant for Ritual Rush.
Pure CastersCan now toggle off Channeler's Stance. Necromancer and Wizard are pure casters — toggle off when you want a different invocation active.

Zone & Item Fixes

  • Najena: Fixed Drelzna's drop rates. Moosh now guaranteed to drop his key. Bloodstained Key no longer NO RENT. Reduced social aggro range for many enemies.
  • Plane of Sky: Inte Akera tuned for solo instances. Dason Goldblade can now properly summon Inte Akera in solo instances. Azaracks/Essence Carriers staked to their island. Quest efreeti no longer assist. Gorgons/Harpies no longer proc mez in solo. Damage numbers adjusted. "a presence" gated from spawning in personal instances.
  • Plane of Fear: Cazic-Thule properly calls minions when engaged in instances. Killing Dread/Fright/Terror/Dracoliche reduces Cazic-Thule's call range.
  • New Sebilisian Expedition: Loom, forge, and merchants added.

UI & Misc Fixes

  • Fixed black screen flicker when entering game from character select
  • Key rings now update properly after loading a loadout
  • Exaltations in Augmentation Storage correctly labeled as Exaltations
  • Fixed floating combat text filtering for misses/dodge/block/etc.
  • Fixed group window context menu — toggle to expand upward instead of downward
  • Fixed negative charge issue with group instance invitations
  • Spell gem swapping fixed — suppressed "already memorized" messages
  • Fixed dead Moth crash

Rogue Changes

  • Stunning Poison adjusted down to level 14
  • Concussive Poison: increased proc chance
